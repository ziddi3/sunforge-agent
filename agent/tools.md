# Planned Agent Tools

These tools describe the intended MongoDB MCP-backed capabilities for the hackathon demo.

## get_ecosystem

Retrieve the root ecosystem record.

## list_repositories

Retrieve sanitized repository metadata.

## list_deployments

Retrieve frontend, backend, and domain deployment records.

## classify_component

Classify a repo, service, document, or asset by role.

## detect_repo_role_drift

Compare a repository's intended role against the responsibilities it is currently carrying.

## check_privacy_boundary

Apply public/private rules and flag content that should remain internal.

## create_task

Create an implementation task from a finding.

## create_audit_report

Write the final ecosystem audit report.

## generate_build_plan

Create a builder-facing next-step plan from the audit report.
